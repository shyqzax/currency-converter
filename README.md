# Конвертер валют

## Описание
Простое веб-приложение для конвертации валют с использованием реального API

## Функционал
- Выбор валют из динамически заполняемых списков
- Конвертация по актуальному курсу
- Сохранение выбранных валют в localStorage
- Кнопка для быстрой смены валют местами

## Технологии
- HTML, CSS, JavaScript
- ExchangeRate API
- Async/await для асинхронных запросов

## Как запустить
1. Открыть index.html в браузере
2. Или посмотреть онлайн: [ссылка на GitHub Pages]
